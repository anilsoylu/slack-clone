## NextJS

### Prerequisites

Node version 20.x

### Cloning the repository

    git clone https://github.com/anilsoylu/slack-clone.git

### Install packages

    npm i

### Setup .env file

    DATABASE_URL=

### Setup Prisma

    npx prisma generate
    npx prisma db push

### Start the app

    npm run dev

### Available commands

Running commands with npm `npm run [command]`

| command | description                              |
| ------- | ---------------------------------------- |
| `dev`   | Starts a development instance of the app |
|         |
